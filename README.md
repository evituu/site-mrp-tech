# MRP Tech - Site Institucional

Este projeto é o site institucional da **MRP Tech**, uma empresa de tecnologia focada em:

- Consultoria em tecnologia para empresas
- Desenvolvimento de softwares sob medida para negócios

## Objetivo

O objetivo deste site é apresentar a MRP Tech, seus serviços, proposta de valor e facilitar o contato com empresas que buscam apoio técnico e soluções de software.

## Desenvolvimento

Este projeto foi iniciado com [Next.js](https://nextjs.org) e está em evolução para atender as necessidades da marca e dos serviços da MRP Tech.

## Como executar localmente

Execute o servidor de desenvolvimento:

```bash
npm run dev
```

Depois, acesse [http://localhost:3000](http://localhost:3000) no navegador.
